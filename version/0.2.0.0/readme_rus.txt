О плагине
---------

Плагин добавляет дополнительную вкладку в "информацию о
пользователе". На вкладке, в компактной форме, отображается вся
доступная информация о пользователе. Сейчас поддержка только
ICQ плагина



Установка
---------

Скопируйте файл look-in.dll в папку с плагинами Миранды

Планируется
-----------

- пока, на поле, в котором представлена информация, нельзя
переключиться с помощью клавиши Tab. Планирую убрать эту фичу

- список строк для перевода (сам плагин уже поддерживает перевод)

- поддержка не только ICQ плагина

- поддержка нескольких ICQ dll'ок (что это означает, я не знаю...)

- переписать плагин на ассемблере, чтобы уменьшить его размер (но мне лень)

- кнопка "обновить" пока не работает

- не работают кнопки OK/Cancel/Apply в настройках

- ещё несколько ошибок с переводом

- поправить формат даты/времени

- ошибки в английском :)
